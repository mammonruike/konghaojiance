#### 运行环境：

python3.8.10以上

## 依赖：
```shell
pip install urllib3
```

## 使用方法：

1、登录或注册阿里云账号（https://www.aliyun.com/）

2、注册服务：https://market.aliyun.com/detail/cmapi00047994?spm=5176.29867242_210807074.0.0.44e83e7eUr5Fgx

3、打开买家控制台找到刚注册的服务复制AppCode并替换脚本中的秘钥。

4、双击脚本用Python打开运行

## 如需打包：

1、安装打包工具：
```shell
pip install auto-py-to-exe
```
2、运行打包工具：
```shell
auto-py-to-exe
```
